## Homework 1 — Local Search: SA, GA & PSO

### Problem Definition

The task is to minimize the following three-variable objective function:

$$f(\mathbf{x}) = \frac{4}{3}\left(x_1^2 + x_2^2 - x_1 x_2\right)^{0.75} + |x_3|, \quad x_1, x_2, x_3 \in [-1, 1]$$

The global minimum is $f^* = 0$, achieved at $\mathbf{x}^* = [0, 0, 0]^T$.

Three algorithms were implemented and compared under a **fair evaluation budget** of **15,750 objective function evaluations** each:

---

## Algorithms

### 🔥 Simulated Annealing (SA)

Neighbors are generated by adding Gaussian noise to the current solution:

$$\mathbf{x}' = \text{clip}(\mathbf{x} + \varepsilon,\ -1, 1), \quad \varepsilon \sim \mathcal{N}(0,\ \sigma^2 I), \quad \sigma = 0.1$$

A **geometric cooling schedule** is used:

$$T_{k+1} = \alpha \cdot T_k, \quad T_0 = 10,\quad \alpha = 0.95$$

with an inner loop of $N_{in} = 50$ iterations per temperature step (~315 outer steps → 15,750 evaluations total).

| Statistic | Value |
|---|---|
| Mean minimum $\bar{f}$ | ≈ 0.002087 |
| Std. deviation $\sigma_f$ | ≈ 0.000912 |
| Best found minimum | ≈ 0.000216 |
| Runtime (avg.) | 4.04 s |

*Results over N = 100 Monte Carlo runs.*

---

### 🧬 Genetic Algorithm (GA)

- **Encoding:** Real-valued (floating-point) vectors $\mathbf{x} \in [-1,1]^3$
- **Selection:** Tournament selection with tournament size $k = 3$
- **Crossover:** Arithmetic crossover — $\mathbf{c}_1 = \alpha\mathbf{p}_1 + (1-\alpha)\mathbf{p}_2$, where $\alpha \sim \mathcal{U}(0,1)$
- **Mutation:** Gaussian perturbation $\varepsilon \sim \mathcal{N}(0, 0.1^2)$ with probability $p_m = 0.1$, applied per individual; result is clipped to domain
- **Elitism:** Best individual always carried forward

Population size: **50**, Generations: **315** → 15,750 evaluations.

| Statistic | Value |
|---|---|
| Mean minimum $\bar{f}$ | ≈ 0.0018 |
| Std. deviation $\sigma_f$ | ≈ 0.0014 |
| Runtime (avg.) | 9.65 s |

*Results over N = 50 Monte Carlo runs.*

---

### 🐦 Particle Swarm Optimization (PSO) *(Bonus)*

Based on the original Kennedy & Eberhart (1995) formulation. Each particle $i$ tracks its current position $\mathbf{x}_i$, velocity $\mathbf{v}_i$, personal best $\mathbf{p}_i$, and the global best $\mathbf{g}$:

$$\mathbf{v}_i^{(t+1)} = w\,\mathbf{v}_i^{(t)} + c_1\mathbf{r}_1 \odot (\mathbf{p}_i - \mathbf{x}_i^{(t)}) + c_2\mathbf{r}_2 \odot (\mathbf{g} - \mathbf{x}_i^{(t)})$$

$$\mathbf{x}_i^{(t+1)} = \mathbf{x}_i^{(t)} + \mathbf{v}_i^{(t+1)}$$

| Parameter | Value |
|---|---|
| Inertia weight $w$ | 0.7 |
| Cognitive / social $c_1 = c_2$ | 2.0 |
| Max velocity $v_{\max}$ | 0.2 |
| Swarm size | 30 particles |
| Iterations | 525 → **15,750 evaluations** |

| Statistic | Value |
|---|---|
| Mean minimum $\bar{f}$ | ≈ 0.000000 |
| Std. deviation $\sigma_f$ | ≈ 0.000000 |
| Runtime (avg.) | **1.39 s** |

*Results over N = 50 Monte Carlo runs.*

## Comparison

All three algorithms were evaluated under identical computational budgets (15,750 function evaluations, 50 MC runs each).

| Algorithm | Mean $\bar{f}$ | Std. $\sigma_f$ | Runtime |
|---|---|---|---|
| **PSO** | **≈ 0.0000** | **≈ 0.0000** | **1.39 s** |
| GA | ≈ 0.0018 | ≈ 0.0014 | 9.65 s |
| SA | ≈ 0.0022 | ≈ 0.0009 | 4.04 s |

**Key takeaways:**
- **PSO** achieved near-exact global optimum consistently across all runs — fastest runtime, lowest variance. Its collaborative update mechanism aggregates global information every iteration, making it exceptionally well-suited for smooth continuous landscapes.
- **GA** reached slightly better median values than SA but with higher variance, indicating sensitivity to parameter tuning (mutation rate, population size).
- **SA** was the most robust of the two classical methods — compact result distribution, predictable behavior, and minimal hyperparameter burden. The geometric schedule with $\alpha = 0.95$ provided a well-balanced exploration-exploitation trade-off.
